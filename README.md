# Overview

**Secura** is an opensource, free Authenticator App for Mobile and Desktop. Wrtten in rust with tauri and React.
Supported Devices are: Windows, Linux (DEB, RPM, AppImage), Android

# Features

Secura is a simple and secure 2FA (Two-Factor Authentication) app. It allows you to:

- [x] Secure login and account encryption
- [x] No Cloud Store required 
- [x] Export accounts to a file or as a QR code
- [x] Add accounts using a name, icon, and QR code or manual input
- [x] Enjoy a clean and user-friendly design
- [x] Use the app on Android and Linux (Windows support is planned)
